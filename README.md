# DrinkUp!

DrinkUp! は、水分補給習慣をシンプルにサポートするアプリです。

コンセプトはとてもシンプルです。  
**誰もが簡単に確実に水分摂取量を把握できること。もっと水を飲もうとする気持ちを育むこと。**

この作品は **2026 Swift Student Challenge** 提出作品として制作しました。

---

## 🧠 コンセプト

日常生活の中で、私たちは水分補給を忘れがちです。
水分補給を意識することも大事ですが、自分がどのくらい飲んでいるか把握できなければきちんと水を飲めているかどうか判断できません。
DrinkUp!ではすべてのユーザーが自分が現時点でどのくらい飲んだのか意識しなくても簡単に記録をつけられるようにすることを目指しました。
そのため、DrinkUp!はユーザーが自分の水筒を用意していることを前提に設計しました。
水筒をもっていない人は用意する必要がありますが、水筒から水を飲むことが最も簡単に記録をつけられる現実的な方法と判断したためです。
DrinkUp! は、達成度を直感的なビジュアルと数字で表現することで、自然に習慣化を促します。

| アイコン | 範囲 | 状態 |
|----------|------|------|
| 🌱 | 0–499ml | まだまだ |
| 🍃 | 500–799ml | いいね |
| 🌳 | 800–1199ml | 素晴らしい |
| 🏆 | 1200ml以上 | 言う事無し |

ユーザーはワンタップで記録できます。
長押しすると任意の値を入力可能です。

本アプリは「短時間の利用」を前提に設計されており、  
記録が終わったらすぐ日常に戻れる体験を重視しています。

---

## 📱 主な機能

- ワンタップで水分記録
- 長押しでカスタム入力
- 「Money Saved」機能
- 1日の履歴表示
- 4段階のAchievement表示
- 明確なナビゲーション
- アクセシビリティ配慮（色のみで識別しない設計）

---

## 🛠 使用技術

### SwiftUI
宣言的UI構築によるシンプルなレイアウト設計。

### SF Symbols
Apple標準アイコンを利用し、Dynamic Typeやアクセシビリティに対応。

### UserDefaults
軽量なローカルデータ保存。

### Combine
日付変更検知や通知処理。

### UIKit連携
- ハプティックフィードバック
- 縦画面固定

これらは、保守性とシンプルさを重視して選択しました。

---

## 🎯 設計思想

- できるだけ短時間で操作を完了
- 装飾的なアニメーションは最小限
- 状態変化は直感的に
- 色以外でも識別できるUI

ユーザーがアプリ内に長く滞在しない設計を目指しています。
